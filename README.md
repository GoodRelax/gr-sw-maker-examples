# gr-sw-maker-examples

Example projects built with [gr-sw-maker](https://github.com/GoodRelax/gr-sw-maker) — a nearly fully automated software development framework.

Each example contains the complete project structure generated by gr-sw-maker, including specs, design documents, source code, tests, and review records.
