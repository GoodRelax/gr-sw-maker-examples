# ユーザーオーダー

## 何を作りたい？

ELM327という Bluetooth 対応OBDドングルで車の診断やログどりをできるPCアプリ

## それはどうして？

車の状態をモニタしたくて、せっかくELM327を買ったのは良いけど、SW書くのが面倒なのでほったらかしていた。
gr-sw-maker ならいい感じのSWを書いてくれそう。

## その他の希望

最低限以下の機能が欲しい。

1. DTC取得 (DoCANと KWP2000)
2. DTC消去 (DoCANと KWP2000)
3. 車速とかエンジン回転数とか一般的な車両データの可視化 (ダッシュボード)
4. 録音ボタンを押したらデータをTSVに記録

ELM327の使い方から、アプリのインストール、使い方まで .md で書いた日本語版マニュアルがいる
車の診断通信についてはネットで調べて

PCは Win11のPC
