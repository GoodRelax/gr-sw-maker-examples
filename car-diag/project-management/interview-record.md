## 1. プロジェクト概要（user-order.md より）

- **What**: ELM327 Bluetooth OBD ドングルで車の診断・ログ取りをする PC アプリ
- **Why**: 車の状態をモニタしたいが SW 作成が面倒だった
- **Platform**: Windows 11

## 2. 機能要求（確認済み）

| # | 機能 | 詳細 |
|---|------|------|
| F1 | DTC 取得 | マルチプロトコル対応: Legacy OBD (Mode 03/07) + UDS (SID $19) + KWP2000 (SID $18)。英語説明文を内蔵データとして搭載。ECU ごとにグループ表示 |
| F2 | DTC 消去 | マルチプロトコル対応: Legacy OBD (Mode 04) + UDS (SID $14) + KWP2000 (SID $14)。全ECU/個別ECU選択可 |
| F3 | 車両データ可視化 | PID 自動検出 + UDS DID スキャン。全検出パラメータをダッシュボードに表示。リアルタイムグラフ（pyqtgraph） |
| F4 | データ記録 | 録音ボタンでサンプリング間隔 500ms で TSV ファイルに記録。リアルタイムグラフ表示と並行して動作 |
| F5 | ECU 自動スキャン | CAN ID スキャン（UDS TesterPresent）+ KWP アドレススキャンで車両上の全ECUを自動検出 |
| F6 | DID スキャン | プリセット（主要DID範囲）+ 設定画面から全スキャン（0x0000-0xFFFF）。中断・再開対応 |
| F7 | スキャン結果キャッシュ | VIN をキーに ~/.car-diag/cache/ にJSON保存。次回接続時は前回結果を再利用。DIDスキャン中断位置も保存 |

## 3. 接続方式

- **Bluetooth のみ**（USB 非対応）
- Windows Bluetooth ペアリング → COM ポート経由で pyserial 接続
- ボーレート: 38400 bps（ELM327 デフォルト）

## 4. UI レイアウト

- **タブ切替方式**: 「接続」「DTC」「ダッシュボード」「記録」の 4 タブ
- ウィンドウ上部にCOMポート選択・接続状態表示
- ウィンドウ下部にステータスバー

## 5. 非機能要求（暗黙）

- デスクトップアプリ（PyInstaller で exe 化）
- 日本語 UI
- 日本語マニュアル（.md 形式）: ELM327 の使い方からアプリのインストール・操作まで

## 6. スコープ外（確認済み）

- USB 接続の ELM327
- OBD-II 以外のメーカー固有プロトコル
- クラウド連携・リモート監視
- 自動アップデート機能
